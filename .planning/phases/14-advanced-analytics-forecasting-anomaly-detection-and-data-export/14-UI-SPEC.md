## Component Specifications

### 1. ForecastChart Client Component

**Location:** `components/analytics/ForecastChart.tsx`

**Visibility rule:** Rendered only when readings count >= 4. If fewer than 4 readings, the parent page renders nothing in the forecast section — no empty state, no message, no placeholder card.

**Container:**
- Full-width `div` with `className="bg-white rounded-[10px] border border-paper-border shadow-card p-6 mt-6"`
- Section heading: `<h2 className="text-[16px] font-semibold text-navy-900 mb-4">Forecast (next 2 periods)</h2>`

**Chart:**
```
ResponsiveContainer width="100%" height={200}
  ComposedChart
    CartesianGrid strokeDasharray="3 3" stroke="#D7E2EF"
    XAxis dataKey="period" tick={{ fontSize: 11, fill: '#3A5270' }}
    YAxis tick={{ fontSize: 11, fill: '#3A5270' }} width={40}
    Tooltip contentStyle={{ fontSize: 12, borderRadius: 6 }}
    ReferenceLine y={targetValue} strokeDasharray="4 2" stroke="#3A5270" strokeOpacity={0.6}
    ReferenceArea y1={forecastLower} y2={forecastUpper} fill="#C8A44A" fillOpacity={0.15}
    Line type="monotone" dataKey="value" stroke={statusColor} strokeWidth={2} dot={false} isAnimationActive={false}
```

**Margins:** `{ top: 8, right: 16, bottom: 8, left: 0 }`

**Data shape passed as prop:**
```ts
{ period: string; value: number | null }[]  // nulls for forecast periods
forecastBand: { lower: number; upper: number }
targetValue: number
status: KpiStatus | IndicatorStatus
```

## Interaction Contracts

### ForecastChart

| Interaction | Behaviour |
|-------------|-----------|
| Hover over chart | Recharts default Tooltip appears — shows period label + actual value. Forecast band periods show "Forecast" as value label. |
| Resize (responsive) | ResponsiveContainer handles — chart reflows, no breakpoints needed |
| Animation | `isAnimationActive={false}` on all Line elements (matches existing KpiSparkline pattern — prevents layout jank) |
| Section visibility | Entire `<section>` element is conditionally rendered. If readings.length < 4, return null from component — no placeholder rendered |

### Analytics Export Table

| Interaction | Behaviour |
|-------------|-----------|
| Download CSV click | `<a>` tag navigation to `/api/analytics/export/[module]` — browser initiates download via `Content-Disposition: attachment` header. No JS required. |
| Download in progress | No loading state needed — browser handles file download natively |
| Download error (HTTP 4xx/5xx) | API returns JSON error — browser shows nothing. Add sonner toast via a client wrapper only if error handling is required. |

### Accordion (API Docs)

| Interaction | Behaviour |
|-------------|-----------|
| Trigger click | shadcn Accordion default expand/collapse animation (CSS-based, no JS configuration needed) |
| Single open | `type="single" collapsible` — only one section open at a time |
| Default state | All accordion items collapsed on page load |
